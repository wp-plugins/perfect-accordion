== Description ==

This plugin will enable Accordion With Custom Post Support in your wordpress theme. You can embed Accordion via shortcode in everywhere you want, even in theme files. 

== Installation ==

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want Accordions in your theme php, Place `<?php echo do_shortcode('[accordion]'); ?>` in your templates.
5. After Activate Accordion U Can See a new menu "Accordion" in your Dashboard Menu. 
6. Than post your text which u want in Accordion.

1. Default Query From  Custom Post
<pre>[accordion]</pre>


2.From Any  Post
<pre>[accordion post_from="post"]</pre>


3.How many post you want
<pre>[accordion count="2"]</pre>
